# Robotic Arm Design using SolidWorks

## Overview
This project involves the design and modeling of a robotic arm using SolidWorks. The robotic arm consists of multiple links, a base, and a gripper assembled to perform pick-and-place operations. The project was developed to gain practical experience in CAD modeling, assembly design, and robotic mechanisms.

## Features
- 3D modeling of robotic arm components
- Complete assembly of all parts
- Gripper end-effector design
- Motion-oriented mechanical structure
- Engineering design using SolidWorks

## Components
- Base
- Link 1
- Link 2
- Link 3
- Link 4
- Link 5
- Gripper
- Final Assembly

## Software Used
- SolidWorks

## Skills Demonstrated
- CAD Modeling
- Assembly Design
- Mechanical Design
- Robotics Fundamentals
- Engineering Drawing Interpretation

## Project Files
- `.SLDPRT` files for individual parts
- `.SLDASM` file for complete assembly
