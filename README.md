# Robotic Arm Design using SolidWorks

## Overview
Designed and modeled a robotic arm using SolidWorks. The project includes individual part modeling, assembly design, and a gripper mechanism. It demonstrates CAD modeling, mechanical design, and robotics fundamentals.

## Features
- 3D CAD modeling
- Multi-link robotic arm assembly
- Gripper end effector
- Mechanical design and assembly

## Software Used
- SolidWorks
